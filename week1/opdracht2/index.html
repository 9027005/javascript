Vragen & Antwoorden

Wat is het verschil met de script tag boven in te laden in de tag t.o.v onder inladen aan het einde van je HTML tag?

als je het boven plaatst moer er een defer bij en anders niet.

Wat doet de attribuut “defer” in de script tag?

Het defer attribuut zorgt er voor dat de browser niet meer blokkeert tijdens het laden van het script en het uitvoeren van het script uitstelt totdat het html is uitgevoerd.

Wat is het verschil tussen HTML en JavaScript?

html is de website zelf en javascript maakt de website interactief.
